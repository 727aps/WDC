# Women's Development Cell – Event Series 2026

This repository contains a static, production-ready website for the Women's Development Cell (WDC) Event Series 2026.  
The site is built using semantic HTML5 and modern CSS3 with a clean, minimal, and mobile-first design suitable for a professional college event listing.

## Project Structure

- `index.html` – Main single-page website for the WDC Event Series 2026.
- `style.css` – Stylesheet containing typography, layout, grid, and card styles.
- `README.md` – Project overview and deployment instructions.

## Technology Stack

- Pure HTML5
- CSS3 (no preprocessors)
- No JavaScript
- No external UI frameworks

## Running the Site Locally

1. Clone or download this repository.
2. Open `index.html` directly in a modern web browser, or serve it with any static file server (for example, using the "Live Server" extension in VS Code).

## Deployment with GitHub Pages

To deploy this site using GitHub Pages:

1. Commit and push the project to a GitHub repository (for example, `https://github.com/727aps/WDC.git`).
2. In the GitHub repository, go to **Settings**.
3. Navigate to the **Pages** section in the sidebar.
4. Under **Source**, select the `main` branch and the root directory (`/`), then save.
5. GitHub Pages will build and publish the site.  
   After a few minutes, your live URL will appear in the GitHub Pages section, and the site will be accessible publicly.
